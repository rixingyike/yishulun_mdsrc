# 墨问终端脚本发布独立版 2.1，支持多图片上传

我对墨问终端脚本进行了一个小小的重构，现在你在下面这个地址可以下载 2.1 版本的 macOS 程序：

[https://github.com/rixingyike/setalias/releases/download/v2.1/SetAlias.zip](https://github.com/rixingyike/setalias/releases/download/v2.1/SetAlias.zip)

解压，会看到三个文件：

![image](assets/clip_20251225-191300.png)

点 install.sh 进行安装，你将看到：

![image](assets/clip_20251225-191339.png)

这是脚本提示你修改 config.sh，所有机密信息，例如 MOWEN_API_KEY 都在这个文件中，修改之后，按回车继续。

![image](assets/clip_20251225-191501.png)

恭喜你，你的终端脚本程序安装成功了！现在你可以用 mnew 在终端里写墨问笔记了，还可以 msummary 总结一周的笔记。

既然是一个 App，你在系统的 Applications 里也可以看到它：

![image](assets/clip_20251225-191623.png)

所有子指令，你点开这个图标再输入，与直接在 iTerm2 或 Terminal 中直接输入，效果是等同的。

酷不酷？！

虽然是一个终端程序，现在长得也像一个独立的 macOS App 了。

之所以这样重构，是为了将脚本、环境常量从系统里抽离出来，达到最小化污染系统的目的。同时因为在安装时有引导，安装也变得简单了。（虽然对程序员来说已经很简单了）

还有一点，你可能已经发现了，虽然我们是在终端里写笔记，但也支持多张图片插入了，你看到的这篇笔记就插入了多张图片。
