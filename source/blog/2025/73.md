# 我感觉现在我无比强大

接上一篇，接着聊 Cursor。我感叹国内没有一款可以平替 Cursor 的 AI IDE，最好是 token 免费的，至少我不用背着梯子使用，可以畅快淋漓地使用。

自从我阅读了 Google Agent 白皮书，我明白了 Agent 是怎么回事。Cursor 每次推出的新功能我都能搞懂，噢，它是这个意思。就说最近 Cursor 更新的 Debug 模式吧，在它更新之前，我自己实际上就在用 “Debug 模式” 修改我的 Bug，Cursor 的更新，只是把我摸索到的修复 Bug 的方法给固化了下来。当然，咱也不能说 Cursor 是抄我的，天下英雄所见略同，只能说，Cursor 工程师和我想到一块去了。

今天上午，大概由于 Design Visually 新功能更新，Cursor 卡得要死，我调用 free auto 模式的速度非常非常慢！我郁闷死了！当时我就意识到，我回不到没有 AI IDE 的过去了，我已经对使用 AI IDE 上瘾了，没有 AI IDE 我不会写代码了吗？不，是不愿意写代码了。自从有了汽车，大街上就看不到骑马的人了。

当时我还萌生另一个想法：给我十台 Apple Air M4，十个 Cursor 不限 token 的 Pro 帐号，再加一条新加载专线，我能用 Cursor 手搓一个平替 Cursor 的 AI IDE 出来！怎么，你觉得不可能？其实没有那么难，理论是公开的，只是 Extension 和 Orchestration 需要仔细调整，而这一块，只要投入足够多的人力、精力就可以。团队我没有，但如果有 10 台 Apple Air M4，每台开 5 个 Cursor，我就相当于有 50 个高级工程师，他们可以不分昼夜辛苦地把活干。我是自然人也需要休息，如果再有一个人与我搭伙，我白天指挥，他晚上指挥，就可以 24 小时连轴工作了。有这样一个团队，一周，一周就可以手搓一个 Cursor 出来！

再说一下墨问吧，这是我经常使用的一个微信生态内嵌笔记软件。它是一个创新性产品，可以说，没有其他任何微信生态内产品比它的粘性强，比它设计得更贴合微信生态。它有一个牛逼的功能，可以将内容转发至微信群内，只有群友可以阅读，这是利用了群 ID 开发的功能；它还有付费功能，可以卖东西，可以分发、传播。一有人评论，在企业服务号上立马就能收到短信通知。它其实就是一个万人微信社区！但它的技术含量其实也没有多高，大概有三台 Apple Air M4，三个 Cursor 帐号，一台机器开一个帐号，三天就能搓一个墨问社区出来了。不过功能要改一改，改成为私域运营服务生意的社区软件，可以发表内容、可以卖东西、粘性强，还有围栏（群内可见），十分适合私域运营使用。这样一款软件，一个拷贝卖十万块钱，绝对 N 多人抢着买。卖十部就是一百万，一百部就是一千万。

bye，不扯了，就扯这么多。我还是想手搓 Cursor 软件，手搓墨问的兴趣不是很大。
