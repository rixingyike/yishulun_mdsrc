# 君子不立危墙之下，别让代码成为呈堂证供

GitHub 是存放代码的圣地，但也可能是职业生涯的 “修罗场”。

## 01 Clowwindy 与 Shadowsocks

时间：2015 年 8 月

如果你入行较早，一定听过 Shadowsocks (SS) 的名字，它是后世无数代理工具的鼻祖。甚至 SS 现在已经成为一种事实协议的名称。

2015 年 8 月 22 日，Shadowsocks 的作者 Clowwindy 突然清空了 GitHub 上的所有代码。随后，他在 Issues 区留下了一段 “遗言”：

> Two days ago the police came to me and wanted me to stop working on this. Today they asked me to delete all the code from GitHub. I have no choice but to obey.（两天前警察来找我，让我停止开发这个项目。今天他们要求我从 GitHub 上删除所有代码。我别无选择，只能照办。）

Clowwindy 仅仅是发明了协议，并未涉及商业化运营，但依然被精准定位。在网络上，没有绝对的隐形人。当影响力大到一定程度，任何人就是透明的。

## 02 逗比根据地的三年刑罚

时间：2017-2018 年

如果说 Clowwindy 只是被 “喝茶”，那么博主 Toyo 的遭遇则是直接的刑事威慑。

Toyo 本名姓孙，运营着一个名为 “逗比根据地” 的网站，他并没有像 Clowwindy 发明新协议，但他编写了大量的一键安装脚本，并撰写了保姆级教程，极大地降低了普通人翻墙的门槛。

2018 年 11 月 14 日孙站长突然失联，其 GitHub 账号 ToyoDAdoubi 被删除，ID 被人抢注。2019 年被警方拘捕；2019 年 3 月 21 日被新密市检察院取保候审，3 月 25 日又被新密市检察院以 “提供侵入计算机系统工具罪” 正式起诉。5 月 31 日被河南省新密市法院以同罪名判处有期徒刑 3 年，缓刑 4 年，罚金两万元，并没收计算机，羁押于河南省新密市看守所。

据调查，孙站长没有真正入狱服满 3 年刑期，根据我国法律，只要在缓刑考验期内不再犯罪，就视为刑罚执行完毕。正常情况下，孙站长的缓刑期在 2023 年左右就已经结束。但他的两万块，和他的爱机应该是真被没收了，还有，他老实了，当年案发之后就隐居于江湖了，现在江湖上查不到孙站长的最新消息了。

## 03 Clash for Windows 删库跑路

时间：2023 年 11 月

2023 年 11 月，Windows 上最流行的客户端 Clash for Windows (CFW) 的作者 Fndroid 突然在 Telegram 频道宣布停止更新：

> 停止更新，也就是大家常说的删库跑路。感谢大家一直以来的支持，有缘江湖再见。

紧接着，Clash Core（核心）的作者 Dreamacro 也删库跑路了。一夜之间，GitHub 上最活跃的 Clash 生态几乎全军覆没。虽无官方通报，但圈内人心知肚明发生了什么。

CFW 虽然只是一个 GUI（图形界面），本身甚至不包含核心协议，但因为它拥有千万级的用户量，它就成为了靶子。用户量越大，风险越大。

## 04 他们真的违法了吗？

整理这些旧闻，并不是为了制造恐慌，而是为了让我自己，及朋友们更多认清法律。也就是普法。

很多开发者，尤其是年轻的朋友，容易陷入一种技术单纯论的误区。觉得我只是写写代码，钻研一下 Rust/Go，研究一下 QUIC 协议，我又没卖钱，没商用，也没反动，到底触犯了什么法律，怕什么？

但现实法律逻辑是明确的。

根据《刑法》第 285 条第 3 款，** 提供专门用于侵入、非法控制计算机信息系统的程序、工具，情节严重的，处三年以下有期徒刑或者拘役。** 而在以往司法实践中，翻墙软件，也就是民间所谓梯子、科学上网，往往被法官直接认定为 == 专门工具 ==。

作为开发者，我们有无数种证明技术能力的方式：可以去给 Linux 内核提 PR，可以去研究分布式数据库，可以去开发高性能的 Web 框架，甚至可以去写个 AI 聊天机器人，没有必要把才华浪费在注定无法见光的灰色地带。

君子不立危墙之下，别让你写的代码，成为损害你自由的呈堂证供。对于梯子或科学上网，不要开发软件，不要分发，不要宣传，知法守法，方能行稳致远。

bye，就聊这么多。作为工程师、程序员、技术人、教育工作者，有时候我们在网上聊相关协议，只是为了研究，增强我们在网络对抗中的实力，期待有朝一日在中美电子对抗中能够报效祖国，只是为了学习而研究，并无他意。
