# 聊一下董事长和猩爷

我在笔记中看到猩爷自己写的 Profile 竟然是 “幼稚、较真”，较真或许有，但幼稚实在看不出来。这可能是猩爷对他人对他评价的一个警示，没错，就是把他人打出的负面评价贴在主页上以警示自己。我看到后，特别想聊一聊董事长和猩爷这两个人。仅从客观层面评价。因为我阅读这两人的笔记已经足够多，我觉得我有足够的评论材料。

猩爷

我认识猩爷是从他幽默风趣的评论中开始的，开始感觉他是一位擅长言语，且可能在生活中会随口说脱口秀的人，时常能给周围带去欢乐。后来有一次读到他写的笔记，老板带他出差，好像去西安，老板可能想让他充当一个行路智慧 + 除天开心果，但猩爷到了西安，对甲方的兴趣都没有对一碗刀削面的兴趣大，就埋头吃面。通过这件事我对猩爷有了巨大改观，猩爷除了幽默，还深负勇气，只要老子不愿意，休想让老子开口。

后来读猩爷的笔记越多，越发现猩爷在公司牛逼，不是投资人就是合伙人，后来又从笔记中得知，他老板欠他钱，他是他老板的债主，牛逼了这事，一般人是做不到的。很多墨友只看到笔记里的猩爷牛逼，但是有些东西大家知道，作者是不写的，这不写的东西就是，在公司里，猩爷身上一定有老板敬佩或离不开的东西，这是猩爷的价值所在。猩爷没写，但我猜到肯定有，不然哪个老板能允许员工骑在自己头上？我断定猩爷有智慧，也是从这个时候开始的。我觉得猩爷有大智慧，实事求是的智慧，务实的智慧，拿捏住老板的智慧，在大城市生存的智慧。

后来我一夸猩爷有智慧，他就谦虚，说我给他戴高帽，其实没有，我一向都是实话实讲。小崔之后，就数我说实话了。

再说一件近期观察到的事，猩爷说年终总结，说来年要打造自己的创作风格，这一点让我困惑。在我看来，猩爷的风格一直都是幽默风趣，他的想法和看问题的角度和一般人不一样，加上 CPU 灵光，他有时候随口拽两句，就幽默尽显。和余华、王小波的作品风格很像。人可能会忽视自己的长处，甚至对其视而不见，猩爷大抵如此。

董事长

再说下董事长。我和董事长吃过饭，线下也算有过交流，又读过不少董事长写的文字了，按说比对猩爷还了解。第一次对董事长有了解，是在极客时间。有一天下午，好像是周末，我去极客时间录音棚录课，我进录音棚前，看见一个中年人靠在一个工程师的工位边上，聊天，对话，也可能是在安排工作。过一段时间，我从录音棚出来了，这人竟然还在那个工位边上聊天，甚至连站着的姿势都没有变化。

当时，我就意识到，这个人可能就是董事长。后来见到董事长真人，真的是他。

在使用墨问笔记以后，还看到过一张董事长站在大师工位旁边的照片，似在聊天、对话。

在 AI 诞生以后，我明白了，那是董事长工作的方法，他在 Debug 他的手下。在 AI 没有诞生之前，董事长就已经深谙 Debug 之道了，他对业务了解之深、思考之完全，加上他灵光的 CPU，导致整个团队中没有人比他更强，他和部下聊天的过程，其实不是在部署工具，而是在借部下的手调试工作，就像我通过 Agent 写代码一样。

就聊这么多吧，bye。如果董事长和猩爷觉得有评价不合适的，可以删除。
