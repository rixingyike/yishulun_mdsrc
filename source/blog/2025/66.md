# Cursor 支持 Design Visually 了。

怎么理解这个功能？我的理解就是它把浏览器及 F12 打开的开发者工具搬到 IDE 里面来了。你若问我喜欢这个功能吗？

不喜欢！至少今天是。大概因为今天发布新功能，今天上午 Cursor 非常非常卡！卡到了模型根本跑不动的程度。当然，有时候别人是好的，因为我用的模型是免费的。

这两天 Cursor 更新非常猛，昨天更新了 Debug 模式，直接拉开了 “一半 AI，一半开发者” 的协作 Agent 模式，今天又把浏览器塞进了 Cursor 里，吾不知 Cursor 意欲何为，兹将一统 AI IDE 江湖吗？

我很郁闷，国内没有一个 IDE—— 免费 token 的 IDE 可以平替 Cursor，如果有，请告诉我。bye，就写这么多。

📅 2025 年 12 月 12 日周五
